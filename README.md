# Pong em JavaScript

Este é um simples projeto de Pong desenvolvido em JavaScript, criado como parte do curso de desenvolvimento web da Alura.

## Descrição

Pong é um dos primeiros jogos eletrônicos lançados e é um clássico atemporal. Este projeto visa recriar a experiência do Pong usando tecnologias web modernas, como HTML5, CSS3 e JavaScript.

## Funcionalidades

- Dois jogadores podem competir entre si, cada um controlando uma raquete.
- A bola rebaterá nas paredes e nas raquetes, mantendo a dinâmica do jogo.
- O placar é mantido e atualizado conforme os jogadores marcam pontos.

## Como Jogar

- Abra o arquivo `index.html` em seu navegador.
- Use as teclas de seta para cima e para baixo para mover a raquete do jogador 1.
- Use as teclas "W" e "S" para mover a raquete do jogador 2.
- O jogador que deixar a bola passar de sua raquete concede um ponto ao oponente.
- O jogo termina quando um jogador atinge a pontuação máxima.

## Pré-requisitos

- Navegador da web moderno que suporte HTML5, CSS3 e JavaScript.

## Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/paulojuniordevs/pong-js.git
   ```

2. Abra o arquivo `index.html` em seu navegador.
